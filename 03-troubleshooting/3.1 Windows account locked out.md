# 1. Problem

User reports:

> "I cannot log into my computer. It says my password is incorrect."

After investigation, the user entered the wrong password multiple times and the account became locked.



# 2. Troubleshooting Process

## Step 1 - Gather Information

Ask:

- When did the issue start?
- Did you recently change your password?
- Is anyone else affected?



## Step 2 - Verify the Issue

Attempt login.

Observe:

> "Account locked out"

or

> "The referenced account is currently locked."



## Step 3 - Identify Root Cause

Check Active Directory:

```
Active Directory Users and Computers
```

User account status:

```
Locked Out = True
```

Root Cause:

> Multiple failed login attempts triggered account lockout policy.



## Step 4 - Resolution

Unlock account.

Reset password if necessary.

Ask user to log in again.



## Step 5 - Verification

Confirm:

- User can log in successfully
