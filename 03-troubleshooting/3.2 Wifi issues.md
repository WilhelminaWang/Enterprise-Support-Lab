# 2. Can not see Network adapter

<img src="./screenshots/image-20260602153242952.png" alt="image-20260602153242952" style="zoom:67%;" />

It means there is a driver issue. 

1. Check the adaptor of another laptop which is the same type with the issue one（If the laptop is connected by Ethernet then check the Ethernet adaptor, if it's connected by Wifi then check the wifi adaptor）.

<img src="./screenshots/image-20260601212253563.png" alt="image-20260601212253563" style="zoom: 67%;" />

<img src="./screenshots/image-20260601212644833.png" alt="image-20260601212644833" style="zoom:50%;" />

2. Then we google: ![image-20260601212755942](./screenshots/image-20260601212755942.png)download the driver, and install it.



# 3. Everything connected but cannot work

For example Lily works in China and she is on a business trip to NZ, she can't access the shared file in NZ. It may be affected by the AD policy, so we need to change her laptop to NZ and apply the policy of NZ.

1. Find the hostname of the client PC

   <img src="./screenshots/image-20260602153843153.png" alt="image-20260602153843153" style="zoom:50%;" />

   <img src="./screenshots/image-20260602153955419.png" alt="image-20260602153955419" style="zoom:50%;" />

2. Go to the server side and find the PC

   <img src="./screenshots/image-20260602154509364.png" alt="image-20260602154509364" style="zoom:50%;" />

   <img src="./screenshots/image-20260602154205114.png" alt="image-20260602154205114" style="zoom:50%;" />

   <img src="./screenshots/image-20260602154343325.png" alt="image-20260602154343325" style="zoom:50%;" />

   we can see the full path of the PC, the PC is under China:

   <img src="./screenshots/image-20260602155243260.png" alt="image-20260602155243260" style="zoom: 50%;" />

3. Move the PC from China to NZ:

   <img src="./screenshots/image-20260602155347669.png" alt="image-20260602155347669" style="zoom:50%;" />

   <img src="./screenshots/image-20260602155416545.png" alt="image-20260602155416545" style="zoom:50%;" />

4. Go the client PC:

   ```
   gpupdate /force
   ```

   then restart the PC
