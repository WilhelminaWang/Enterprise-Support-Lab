# Enterprise IT Support Lab

# 1. Overview

This project simulates a real-world enterprise IT support environment using Microsoft Entra ID and Microsoft Intune.

It demonstrates core IT support operations including:

- User lifecycle management
- Device enrollment and endpoint management
- Common troubleshooting scenarios in an enterprise environment

The goal of this project is to showcase practical IT support, Microsoft 365 administration, and troubleshooting skills relevant to modern workplace IT roles.



# 2. Objectives

- Simulate enterprise-level IT support operations
- Practice Microsoft Entra ID (Azure AD) identity management
- Deploy and manage devices using Microsoft Intune
- Document real-world troubleshooting scenarios
- Build a structured IT support knowledge base



# 3. Technologies Used

- Microsoft Entra ID (Azure Active Directory)
- Microsoft Intune
- Microsoft 365 (Developer Tenant)
- Windows 10/11 Virtual Machine
- Basic networking concepts



# 4. Project Structure

## 4.1 User Lifecycle Management

- User creation and deletion
- Group assignment
- Role-based access simulation



## 4.2 Device Management

- Device enrollment into Intune
- Compliance policies
- Configuration profiles



## 4.3 Troubleshooting Scenarios

- Outlook sync issues
- MFA authentication problems
- Device enrollment failures



## 4.4 Screenshots

All evidence and screenshots are stored in the `/image` folder and referenced within each documentation file.



# 5. Outcomes

- Microsoft Entra ID identity management
- Endpoint management using Intune
- Enterprise IT troubleshooting methodology
- Incident-based documentation (ITIL-style thinking)
- Structured technical communication
